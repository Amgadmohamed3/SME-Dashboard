# 📈 SME Dashboard — Power BI

## 📋 Project Overview

A Power BI dashboard focused on SME (Small & Medium Enterprise) lending analytics. The dashboard tracks key portfolio metrics including loan disbursement volumes, repayment rates, delinquency trends, and relationship manager performance — supporting data-driven credit decisions.

**Tech Stack:** Power BI, DAX, Data Modeling

---

## 🎯 Objective

- Monitor SME loan portfolio health and performance metrics
- Track relationship manager productivity and tenure within the company
- Visualize delinquency patterns and collection effectiveness
- Support management reporting with interactive drill-down capabilities

---

## 📊 Dashboard Components

The `.pbix` file contains the full data model, DAX measures, and multi-page report.
